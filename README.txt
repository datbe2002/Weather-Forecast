-Usage

+Enter a City Name: In the search box, type the name of the city for which you want to check the weather forecast.

+Click Enter: Press Enter to retrieve the weather forecast for the specified city.

+View Weather Details: The app will display the current weather details for the city, including temperature, humidity, wind speed, and weather conditions.

+5-day Forecast: Below the search box , you will find a 5-day forecast showing the weather forecast for each day, starting from the current date.

+You can click on each day to see detail.

-Technologies Used

+React: A JavaScript library.
+OpenWeatherMap API: An API that provides weather data.
